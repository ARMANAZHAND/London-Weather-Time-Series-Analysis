# London Weather Time Series Analysis
### Arman Azhand

## Overview

A time series analysis in R on London weather. For the purpose of my analysis, I decided to focus on the mean temperature (TG) and look at some other features such as precipitation (RR), humidity (HU), and sea level pressure (PP). A comprehensive report in R Markdown has been generated, and can be viewed via the appropriate HTML file. A powerpoint presentation with methodologies further explained in a concise manner is also readily available.

## Data

This dataset (from [Kaggle](https://www.kaggle.com/datasets/zongaobian/london-weather-data-from-1979-to-2023)) provides daily weather observations from Heathrow, United Kingdom (STAID: 1860), spanning the period from 1979 to 2023. The data is sourced from the European Climate Assessment & Dataset (ECA&D) and includes multiple weather parameters such as temperature, precipitation, sunshine, and more.
